    Nvm管理Node.js的版本

    Merry He 2/10/2022

    企业的项目中，有些项目比较老可能使用的低版本的Node.js, 而有一些新项目可能又需要高版本的Node.js， 并行开发时， 频繁的卸载和安装Node显然很麻烦， 可以使用 nvm 来快速切换Node.js的版本， 以适应不同的项目。

    # 什么是nvm

    nvm（node.js version management）， 是一个Nodejs的版本管理工具，为了解决Node.js各种版本存在不兼容现象， 可以通过它可以安装和切换不同版本的Node.js。

    # nvm下载和安装

    安装 nvm 前，如果已经下载了Node.js, 需要先卸载掉;

    • nvm windows下载地址 ： https://github.com/coreybutler/nvm-windows/releases
    • 选择 nvm-setup.zip 下载压缩包，下载到本地之后双击压缩包里的 nvm-setup.exe 进行安装,在安装时可选择 nvm 安装路径和 Node.js 路径 ； 安装完成后打开 cmd 控制面板，执行 nvm -v 显示版本号表示安装成功

    # 设置镜像

    在安装 nvm 时选择的目录下找到 settings.txt, 在文件末尾加入以下代码设置阿里云镜像，加快国内访问速度。

    node_mirror: https://npm.taobao.org/mirrors/node/
    npm_mirror: https://npm.taobao.org/mirrors/npm/

    # nvm基本使用

    • nvm list available      显示 Node.js 可下载版本的部分列表
    • nvm install 版本号    安装指定的版本的Node.js (例: nvm install 18.18.0 ，执行命令后可使用 node -v 查看是否安装成功)
    • nvm use 版本号      使用指定版本的Node.js, 即切换Node版本 (例：nvm use 18.18.0)
    • nvm list        查看目前已经安装的 Node.js 版本, 带 * 号的表示当前正在使用的版本
    • nvm uninstall 版本号     删除某个已安装的 Node.js 版本
